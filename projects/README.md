I spent my weekend building a Python tool that analyzed 2 months of my hostel WhatsApp group activity.

Some interesting findings:

• Analyzed 3,174 messages from 6 group members over 60 days
• Identified the group's most active members and message share percentages
• Found the busiest day (04 May 2024) and busiest hour (6 PM – 7 PM)
• Calculated response speeds to identify the fastest and slowest repliers
• Detected inactivity streaks and group "ghosts" automatically
• Generated personality archetypes such as:

* THE NIGHT OWL
* THE GHOST
* THE STORYTELLER
* THE DRAMA QUEEN
* THE QUESTION MASTER
* THE SPAMMER

Top group words:
"guys" (287), "bhai" (160), "yaar" (139), "kya" (133)

The project was built using only Python fundamentals and NumPy.

No pandas.
No matplotlib.
No regex.

Just:
• Loops
• Dictionaries
• Strings
• Functions
• File Handling
• One NumPy matrix for the activity heatmap

Features:
✓ Message statistics
✓ Member ranking
✓ Activity analysis
✓ Response time analysis
✓ Inactivity tracking
✓ Personality classification
✓ Word frequency analysis
✓ Activity heatmaps

A fun project that helped me practice real-world data analysis while strengthening my Python fundamentals.
